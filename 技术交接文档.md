# DeepSeek中转服务 - 技术交接文档

## 📋 项目简介

DeepSeek中转服务是一个纯Java实现的自然语言转键盘指令的转换模块，将用户自然语言输入转换为结构化键盘操作（按键组合或剪贴板粘贴）。

**目标：
- 用户输入：自然语言（如"把前三个字加粗"）
- DeepSeek输出：结构化Actions数组
- 客户端：将Actions交给现有的蓝牙HID键盘逻辑

## 🔧 集成步骤

### 1. 复制代码包

```bash
# 将 deepservice/ 包复制到安卓项目的
# app/src/main/java/ 目录下
```

### 2. 添加依赖

在 `app/build.gradle` 中添加：

```gradle
dependencies {
    // JSON解析库
    implementation 'com.google.code.gson:gson:2.10.1'
    
    // HTTP请求库（推荐）
    implementation 'com.squareup.okhttp3:okhttp:4.12.0'
}
```

### 3. 配置API Key

方式：

#### 方式A：在 `local.properties`

```properties
# 根目录的 local.properties
deepseek.api.key=sk-xxxxxxxxxxxxxxxx
```

#### 方式B：build.gradle 配置

```gradle
android {
    defaultConfig {
        // 读取 local.properties
        Properties properties = new Properties()
        properties.load(project.rootProject.file("local.properties").newDataInputStream())
        
        // 配置
        buildConfigField "String", "DEEPSEEK_API_KEY", "\"${properties.getProperty("deepseek.api.key", "")}\""
    }
}
```

## 💻 代码集成示例

### 完整使用示例

```java
import deepservice.DeepSeekClient;
import deepservice.exception.DeepSeekAPIException;
import deepservice.exception.ActionParseException;
import deepservice.model.Action;
import deepservice.model.ActionType;
import deepservice.model.Platform;

public class MyActivity extends AppCompatActivity {

    private DeepSeekClient deepSeekClient;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        
        // 初始化客户端
        initDeepSeekClient();
    }

    private void initDeepSeekClient() {
        try {
            deepSeekClient = new DeepSeekClient(
                BuildConfig.DEEPSEEK_API_KEY,
                Platform.WINDOWS // 或 Platform.MAC
            );
        } catch (Exception e) {
            e.printStackTrace();
        }
    }

    // 处理用户输入
    public void onUserInput(String userInput) {
        try {
            // 调用DeepSeek API
            List<Action> actions = deepSeekClient.execute(userInput);
            executeActions(actions);
        } catch (DeepSeekAPIException e) {
            Toast.makeText(this, "API调用失败：" + e.getMessage(), Toast.LENGTH_SHORT).show();
        } catch (ActionParseException e) {
            Toast.makeText(this, "解析失败：" + e.getMessage(), Toast.LENGTH_SHORT).show();
        } catch (Exception e) {
            Toast.makeText(this, "错误：" + e.getMessage(), Toast.LENGTH_SHORT).show();
        }
    }

    // 执行Actions（交给现有蓝牙HID逻辑
    private void executeActions(List<Action> actions) {
        for (Action action : actions) {
            switch (action.getType()) {
                case KEY_COMBO:
                    // 调用现有蓝牙HID发送按键
                    List<String> keys = action.getKeys();
                    sendHidKeyCombo(keys);
                    break;

                case CLIPBOARD_PASTE:
                    // 调用现有剪贴板同步和粘贴逻辑
                    String text = action.getText();
                    syncClipboardAndPaste(text);
                    break;
            }
        }
    }

    // 调用您现有的蓝牙HID发送
    private void sendHidKeyCombo(List<String> keys) {
        // 在此调用您原有的HID发送方法
        // 例如：HIDService.sendKeys(keys);
    }

    // 调用您现有剪贴板同步和粘贴逻辑
    private void syncClipboardAndPaste(String text) {
        // 在此调用您原有剪贴板方法

}
```

## 📊 数据结构

### Action对象结构

```java
// Action.java - 单个操作对象
public class Action {
    private ActionType type;        // KEY_COMBO 或 CLIPBOARD_PASTE
    private List<String> keys;    // 按键组合（仅KEY_COMBO有）
    private String text;        // 粘贴文本（仅CLIPBOARD_PASTE有）
    private String description; // 操作描述
}
```

## 🔍 故障排查

### 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| API Key错误 | Key无效或过期 | 检查DeepSeek控制台检查并更新 |
| JSON解析失败 | DeepSeek返回格式不对 | 查看 [SystemPrompt提示词.md](file:///workspace/SystemPrompt提示词.md) 规则 |
| 网络超时 | max_tokens太小 | 已设为5000 |
| 快捷键选错 | 提示词规则太旧 | 确认是否用最新版Prompt |

## ⚠️ 注意事项

1. **不需要打包成额外的东西！直接复制源代码集成。
