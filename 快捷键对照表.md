# 键盘快捷键对照表

## Windows vs macOS 快捷键对比

| 操作 | Windows | macOS |

| 全选 | Ctrl+A | Cmd+A |
| 复制 | Ctrl+C | Cmd+C |

| 剪切 | Ctrl+X | Cmd+X |

| 删除 | Delete | Backspace |
| 换行 | Enter | Enter |
| Tab | Tab | Tab |

## 本次测试用的快捷键回顾

### TC001: 删除全文

- Windows: Ctrl+A → Delete
- macOS: Cmd+A → Backspace

### TC002: 放大前三个字

- Windows: Shift+Left × 3 → Ctrl+Shift+> (增大字号)
- macOS: Shift+Left × 3 → Cmd+Shift+> (增大字号)

### TC003: 加粗前三个字

- Windows: Shift+Left × 3 → Ctrl+B
- macOS: Shift+Left × 3 → Cmd+B
